# **App Name**: TempleConnect

## Core Features:

- Temple Registration: Allow temples to register and manage their profiles.
- Pooja Listing: Enable temples to list available poojas with descriptions and schedules.
- Donation System: Allow users to donate to temples through secure payment gateways.
- Product Sales: Enable temples to sell prasad and religious items online.
- Secure Payment Integration: Integrate Razorpay for secure payments for pooja bookings, donations, and product purchases.
- User Subscription Management: Manage user subscriptions with free and premium plans for discounts, automating status updates within the app.
- Automated Notifications: Send automated confirmation emails using SMTP or Nodemailer after bookings and donations.

## Style Guidelines:

- Primary color: Saffron (#FF9933) to evoke spirituality and tradition.
- Background color: Beige (#F5F5DC), a desaturated version of saffron, for a calm, clean feel.
- Accent color: Gold (#D4AF37), analogous to saffron, to highlight key actions and elements.
- Body and headline font: 'Alegreya', a serif, provides an elegant, intellectual feel for both headers and content.
- Use traditional Hindu symbols and motifs in the icons.
- Clean and intuitive layout with clear sections for temples, poojas, donations, and products.
- Subtle transitions and animations to enhance user experience during bookings and transactions.
